# Informed Masculinity Website

A modern, professional single-page website for the 501(c)(3) nonprofit **Informed Masculinity**.

## 🌟 Features

- **Hero section** with strong tagline and CTAs
- **About section** with mission, vision, and impact statistics
- **The Puberty Box** — flagship interactive modal with three age-specific tracks:
  - Ages 10–11: Foundations of Change
  - Ages 12–13: Navigating the Surge  
  - Ages 14–15: Owning Your Future
- **Resources section** with direct links to:
  - [HeadsUpGuys.org](https://headsupguys.org/)
  - [Talk With Turek Podcast](https://www.theturekclinic.com/talk-with-turek-podcast/)
  - [Man Enough Podcast](https://manenough.com/podcast/)
- Fully responsive design (Tailwind CSS)
- Donation and enrollment modals
- Professional nonprofit aesthetic (navy + teal)

## 🚀 Quick Start

1. Download or clone this repository
2. Open `index.html` directly in any browser (no build step required)
3. Customize text, colors, and images as needed

## 📦 Deployment Options

### GitHub Pages (Free & Recommended)
1. Create a new public repository on GitHub named `informed-masculinity` (or your preferred name)
2. Upload `index.html` and `README.md`
3. Go to **Settings → Pages**
4. Set Source to **Deploy from a branch** → `main` / `(root)`
5. Your live site will be at: `https://yourusername.github.io/informed-masculinity`

### Other Easy Options
- **Netlify Drop**: Drag & drop the folder
- **Vercel**: Import the repo
- **Cloudflare Pages**: Free and fast

## 🛠 Customization Tips

- Replace placeholder images with real photos (use picsum or your own)
- Update the 501(c)(3) details and EIN
- Connect the donation button to Stripe or your preferred processor
- Add your logo in the navbar
